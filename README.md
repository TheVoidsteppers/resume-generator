# resume-generator
个人简历生成 pdf

## Getting Started

### Installation

To use Puppeteer , run

```shell
npm install
```

### DEVELOP

```shell
# change directory to server , run 
node index.js # start node server
# change directory to current poject root directory ,  run 
npm run dev
```

PRODUCT

```shell
# change directory to current poject root directory ,  run 
npm run pro
```
